# SoftwareArchitecture-Project

<h2>How to run?</h2>
<ol>
  <li>docker-compose up --build -d</li>
  <li>New terminal, then: cd frontend, open index.html</li>
</ol>
<h2>How to terminate?</h2>
<ol>
  <li>Ctrl C</li>
  <li>docker-compose down</li>
</ol>
