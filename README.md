# SoftwareArchitecture-Project

<h1>How to run?</h1>
<ol>
  <li>docker-compose up -d<li>
  <li>cd monitor</li>
  <li>node server.js</li>
  <li>New terminal, then: cd frontend, open index.html<li>
</ol>
<h1>How to terminate?</h1>
<ol>
  <li>Ctrl C</li>
  <li>docker-compose down</li>
</ol>
